# 👕 ReWear – Community Clothing Exchange

A MERN-based platform that promotes sustainable fashion by enabling users to exchange, discover, and reuse pre-owned clothing within a community.

## 🚀 Features

* User authentication and profile management
* Create and manage clothing listings
* Image upload and media management
* AI-powered semantic search using Gemini API
* Automated product description generation
* Community-driven clothing exchange system

## 🛠️ Tech Stack

**Frontend:** React.js, HTML, CSS, Bootstrap

**Backend:** Node.js, Express.js

**Database:** MongoDB

**AI:** Gemini API, Semantic Search

**Tools:** Git, GitHub, Postman

## 📌 Highlights

* Developed core marketplace functionality in a 3-member team
* Designed backend architecture and database schema
* Integrated AI-powered search and content generation
* Built scalable APIs for managing users and listings

## ⚙️ Setup

```bash
git clone <repository-url>
npm install
npm run dev
```

Create a `.env` file:

```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret
GEMINI_API_KEY=your_api_key
```

## 🌱 Goal

ReWear encourages sustainable clothing reuse by connecting users through a community-driven exchange platform.
